<h2>Overview</h2>
<p>Faulty-Calculator is an interactive web-based tool designed to simulate calculation errors, enabling developers to test and improve error handling and system resilience. By intentionally introducing probabilistic faults into calculations, it provides a practical environment for evaluating fault tolerance within larger applications.</p>
<p><strong>Why Faulty-Calculator?</strong></p>
